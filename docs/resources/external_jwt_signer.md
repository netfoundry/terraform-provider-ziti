# ziti_external_jwt_signer (Resource)

Ziti External jwt signer Resource

## Example Usage

```terraform

  name              = "test_external_jwt_signer"
  scopes            = ["test"]

  issuer            = "test1"

  cert_pem          = "-----BEGIN CERTIFICATE-----\naaaaaaaaa\nbbbbbbb\n-----END CERTIFICATE-----\n"

  }
}
```
